# Tuff Mobile App

A premium Flutter-based fitness application designed to provide a seamless user experience for both members and administrators.

> [!IMPORTANT]
> **Project Status**: The primary focus was on implementing a high-fidelity UI, intuitive navigation flows, and a robust Admin Dashboard structure. While the core interface is functional, some deeper business logic and secondary features are currently placeholders.


## Navigation and Access

To explore the different parts of the application, use the following credentials on the login screen:

### Admin Access
To navigate to the **Admin Dashboard**:
- **Email:** `admin@gmail.com`
- **Password:** `123`

### User Access
For any other credentials or input, the application will navigate directly to the **Home Page**.

## Instructions to Run the Application

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed on your machine.
- A code editor (e.g., VS Code or Android Studio).
- An Android/iOS emulator or a physical device.

### Setup and Execution

1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   cd tuff_mobile_app
   ```

2. **Install Dependencies**:
   ```bash
   flutter pub get
   ```

3. **Run the Application**:
   ```bash
   flutter run
   ```
